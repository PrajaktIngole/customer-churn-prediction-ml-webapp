# 🚀 Customer Churn Analysis & Prediction System

An end-to-end Customer Churn Analytics and Machine Learning Prediction system built using:

- MySQL (Data Storage & SQL Analysis)
- Python (Data Cleaning & ML Model)
- Scikit-learn (Logistic Regression)
- Power BI (Interactive Dashboard)
- Flask (Web Application)
- Render (Cloud Deployment)

---

## 📌 Project Overview

Customer churn is a major problem for telecom companies. This project analyzes customer behavior and builds a Machine Learning model to predict the probability of churn.

The system includes:

- SQL-based exploratory analysis
- Data preprocessing & feature engineering
- ML model training (Logistic Regression)
- Interactive Power BI dashboard
- Deployed web app for real-time churn prediction

---

## 🧠 Machine Learning Model

- Algorithm: Logistic Regression
- Accuracy: ~79.6%
- Evaluation Metrics:
  - Precision
  - Recall
  - F1-Score
  - Confusion Matrix

The model predicts churn probability and classifies customers as:
- Low Risk
- Medium Risk
- High Risk
